# Profile-Changer
Android application for changing profiles on receiving a text message.

## Installation

**1. Fork and Clone this repo:**

```sh
git clone https://github.com/YOUR_USERNAME/Profile-Changer.git
cd Profile-Changer
```

**2. Create branch:**
```sh
git checkout -b branch-name
```
